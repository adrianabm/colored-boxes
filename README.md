# Colored Boxes

## Getting Started

1. Clone the repository
  ```
  git clone git@github.com:adrianabm/colored-boxes.git
  ```

2. Enter the project directory
  ```
  cd colored-boxes
  ```

3. Install all dependencies
  ```
  npm install
  ```

4. Run the application - it will be available on http://localhost:8080
  ```
  npm start
  ```
